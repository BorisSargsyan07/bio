CSS

CSS կամ կասկադային ոճաթերթ (Cascading Style Sheets), կայքին ձևավորում տալու համար օգտագործվող տեխնոլոգիա ծրագրավորման ֆորմալ լեզվով։ CSS-ը հիմնականում օգտագործվում է HTML և XHTML ֆորմատի պատկանող վեբ-էջերի ձևավորման համար, սակայն կարող է նաև օգտագործվել ցանկացած տեսակի XML ֆորմատի փաստաթղթերի համար, ներառյալ SVG և XUL, ինչպես նաև դիզայնի, մակետի, տարբեր սարքերի և տարբեր էկրանների վրա ներկայացնելու համար: Ավելին տեսնելու համար սեղմեք ԱՅՍ ԼԻՆՔԻ ՎՐԱ։

CSS-ի առավելությունները

  • CSS-ը խնայում է ժամանակը −Դուք կարող եք մեկ անգամ գրել CSS, ապա նորից օգտագործել նույն թերթիկը բազմաթիվ HTML էջերում: Դուք կարող եք սահմանել ոճ յուրաքանչյուր HTML տարրի համար և կիրառել այն այնքան վեբ էջերում, որքան ցանկանում եք:
  • Էջերն ավելի արագ են բեռնվում – Եթե դուք օգտագործում եք CSS, ձեզ հարկավոր չէ ամեն անգամ գրել HTML պիտակի ատրիբուտներ: Պարզապես գրեք պիտակի մեկ CSS կանոն և կիրառեք այն այդ թեգի բոլոր երևույթների վրա: Այսպիսով, ավելի քիչ կոդը նշանակում է ավելի արագ ներբեռնման ժամանակ:
  • HTML-ին գերազանցող ոճեր – CSS-ն ունի ատրիբուտների շատ ավելի լայն զանգված, քան HTML-ը, այնպես որ դուք կարող եք շատ ավելի լավ տեսք տալ ձեր HTML էջին՝ համեմատած HTML ատրիբուտների:
  • Համաշխարհային վեբ ստանդարտներ – Այժմ HTML ատրիբուտները հնացած են, և առաջարկվում է օգտագործել CSS: Այսպիսով, լավ գաղափար է սկսել օգտագործել CSS-ը բոլոր HTML էջերում, որպեսզի դրանք համատեղելի լինեն ապագա բրաուզերների հետ:
